Installation of ScanHawk Desktop

Note: ScanHawk Desktop is only available for Windows systems. Additional hardware is required.

ScanHawk Desktop is the desktop program used to power the ScanHawk system. With it you can register attendance using NFC.

Below is a guide on how to install the program. If you need help with usage, see ScanHawk Desktop. If you need help with the full system, see ScanHawk Enterprise.

Step 1: Downloading

To install ScanHawk Desktop, visit our downloads page.

Then you just find your preferred version, and click the correct download option as a .zip file.

Step 2: Installation

Now you should have a .zip file downloaded to your computer. The next steps are as follows:

  • Extract the .zip file to a location on your computer.
  • You will now have multiple files inside a folder, including an exe file.

That's it. Now everything you need is located withing this unzipped folder. To run the program, just open the program file. If you want to host a webserver with the PHP file located in the same folder, you may need to use a third-party service or program like XAMPP or something similar.
