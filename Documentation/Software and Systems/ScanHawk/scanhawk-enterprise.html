Overview

ScanHawk Enterprise is a smart NFC-based attendance registration system, designed for businesses, schools, and other institutions that need a reliable and efficient tracking solution.

Components

  • NFC Reader: Detects and registers NFC cards.
  • Display: Provides real-time feedback to users.
  • Windows-based Computer: Runs ScanHawk Desktop, handling all logic and data processing.

System Requirements

  • Operating System: Windows 10 or later
  • Processor: Intel i3 or equivalent (64-bit required)
  • RAM: Minimum 4GB (8GB recommended)
  • Storage: At least 1GB free space
  • Network: A connection to the local network is required for some features like live log updates.

Installation

To install ScanHawk Enterprise, follow these steps:

  1. Ensure your system meets the minimum requirements.
  2. Download and install ScanHawk Desktop on your Windows computer.
  3. Connect the NFC reader via USB.
  4. Run ScanHawk Desktop and follow the on-screen setup instructions.

This only applies to the demo version. If you are using the final release version, please see the complete documentation.

Using ScanHawk Enterprise

Registering NFC Cards

To register an NFC card, scan the card using the reader while ScanHawk Desktop is running. The software will prompt you to enter user details.

Checking Attendance

Users can scan their registered NFC cards, and their attendance will be logged automatically.

Viewing Attendance Logs

Attendance records can be viewed and exported through the ScanHawk Desktop interface.
